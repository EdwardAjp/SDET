# Playwright Login Automation Tests

## Overview
This project is a Playwright-based test automation framework created as part of an SDET take-home assessment.  
It validates the login functionality of https://the-internet.herokuapp.com/login using positive and negative test scenarios.

## Tech Stack
- Playwright
- JavaScript (Node.js)
- Page Object Model (POM)
